# 第2章 练习2.1 - Lean形式化证明

## 题目

**练习2.1**：证明，对于任何对称矩阵 A，问题 max_{U ∈ O(D,d)} tr(U^T A U) 的解是矩阵 U*，其列是 A 的前 d 个单位特征向量。

## 问题背景

这是主成分分析(PCA)的核心数学定理，说明了为什么PCA通过计算协方差矩阵的特征向量来找到最优的低维表示。

### 符号说明
- A ∈ ℝ^{D×D}: 对称矩阵（通常是数据的协方差矩阵）
- U ∈ O(D,d): 列正交矩阵，即 U^T U = I_d
- tr(·): 矩阵的迹（对角元素之和）
- λ₁ ≥ λ₂ ≥ ... ≥ λ_D: A的特征值（降序排列）
- v₁, v₂, ..., v_D: 对应的单位特征向量

## 证明思路

### 第一步：目标函数的重新表述

对于列正交矩阵 U = [u₁, u₂, ..., u_d]（其中 uᵢ 是列向量），我们有：

```
tr(U^T A U) = tr([u₁^T; u₂^T; ...; u_d^T] A [u₁, u₂, ..., u_d])
            = Σᵢ₌₁ᵈ uᵢ^T A uᵢ
```

因此，优化问题变为：
```
max Σᵢ₌₁ᵈ uᵢ^T A uᵢ
s.t. uᵢ^T uⱼ = δᵢⱼ (正交性约束)
```

### 第二步：瑞利商（Rayleigh Quotient）

对于单位向量 v（即 ‖v‖ = 1），二次型 v^T A v 称为瑞利商。

**关键性质**（来自谱定理）：对于对称矩阵 A，
```
λ_min ≤ v^T A v ≤ λ_max
```
且当 v 等于对应的特征向量时取得等号。

### 第三步：Courant-Fischer极小极大定理

这是证明的核心工具。定理指出：

对于 k 个正交的单位向量 v₁, ..., v_k，
```
Σᵢ₌₁ᵏ vᵢ^T A vᵢ ≤ Σᵢ₌₁ᵏ λᵢ
```
且当 vᵢ 等于前 k 个特征向量时取得等号。

**证明提示**：
1. 考虑 A 在正交向量 v₁, ..., v_k 张成的子空间上的限制
2. 这个限制矩阵的特征值不超过 A 的前 k 个特征值（变分原理）
3. 迹等于特征值之和

### 第四步：应用到我们的问题

由于 U 的列 u₁, ..., u_d 是正交的单位向量，根据第三步：
```
tr(U^T A U) = Σᵢ₌₁ᵈ uᵢ^T A uᵢ ≤ Σᵢ₌₁ᵈ λᵢ
```

当 U* = [v₁, v₂, ..., v_d]（前 d 个特征向量）时：
```
tr(U*^T A U*) = Σᵢ₌₁ᵈ vᵢ^T A vᵢ = Σᵢ₌₁ᵈ λᵢ
```
因为 A vᵢ = λᵢ vᵢ。

因此 U* 是全局最优解。□

## Lean形式化的结构

### 主要定义

1. **IsColumnOrthogonal**: 列正交矩阵的定义
2. **IsSymmetric**: 对称矩阵的定义
3. **RayleighQuotient**: 瑞利商的定义

### 关键引理

1. **trace_cyclic**: 迹的循环性质
2. **objective_as_quadratic_forms**: 目标函数展开为瑞利商之和
3. **rayleigh_quotient_bounded_by_max_eigenvalue**: 单个瑞利商的界
4. **sum_rayleigh_quotients_bounded**: 多个正交向量的瑞利商之和的界（核心引理）

### 主定理

**pca_optimality**: 证明前 d 个特征向量最大化迹

### 推论

**pca_uniqueness_up_to_rotation**: 在特征值不重复时，最优解唯一（模正交旋转）

## 与书中内容的联系

这个定理是第2章（学习线性和独立结构）的基础，它：

1. **理论基础**：解释了为什么PCA能找到最优的低维子空间
2. **算法动机**：导出了幂迭代等计算方法（见第2章第297-346行）
3. **几何直觉**：主成分对应数据方差最大的方向
4. **统计意义**：当A是协方差矩阵时，U*给出最小重构误差

在后续章节中，这个结果将被推广到：
- 核PCA（非线性低维结构）
- 概率PCA（带噪声的生成模型）
- 深度自编码器（非线性PCA的神经网络版本）
