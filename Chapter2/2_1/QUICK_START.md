## 📌 一分钟了解

**定理**：对称矩阵A的最优投影子空间由其前d个特征向量确定

**公式**：max_{U∈O(D,d)} tr(U^T A U) 在 U*=[v₁,...,v_d] 取最大值

**意义**：这是PCA（主成分分析）的数学基础

## 🚀 五分钟速成

### 证明核心思路

```
1. 目标 = Σᵢ uᵢ^T A uᵢ    (展开迹)
2. uᵢ^T A uᵢ ≤ λᵢ          (瑞利商)
3. Σᵢ uᵢ^T A uᵢ ≤ Σᵢ λᵢ    (求和)
4. 特征向量取等号          (最优)
```
